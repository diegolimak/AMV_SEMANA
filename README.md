# 📊 Reunião Semanal — Corretora Amor à Vida

Dashboard interativo de apresentação para as reuniões comerciais semanais da **Corretora Amor à Vida (AMV)**.

## 🗂️ Estrutura do Projeto

```
Reunião Semanal/
├── index.html          # Dashboard principal (visão geral / apresentação)
├── admin.html          # Painel administrativo com dados detalhados
├── assets/
│   ├── logo-horizontal-branca.png
│   ├── logo-horizontal-colorida.png
│   ├── logo-simbolo-branco.png
│   ├── logo-simbolo-colorido.png
│   └── photos/         # Fotos da equipe/corretora
├── css/                # Estilos customizados
└── js/
    ├── chart.umd.min.js  # Biblioteca Chart.js (local)
    └── data.js           # Dados da reunião (métricas, metas, resultados)
```

## 🚀 Como Usar

Basta abrir o arquivo `index.html` no navegador — não requer instalação ou servidor.

> Para apresentar em reunião: abra `index.html` em tela cheia (F11).

## 📝 Atualizando os Dados

Edite o arquivo `js/data.js` para atualizar as métricas de cada semana:
- Resultados de vendas
- Metas vs. realizado
- Ranking de corretores
- Indicadores de desempenho

## 🏢 Sobre a AMV

**Corretora Amor à Vida** é uma corretora de planos de saúde com foco em atendimento humanizado e gestão 360° para empresas e pessoas físicas.
